# Raylib 2D RPG

2D RPG built using raylib. Tutorial from: https://www.youtube.com/watch?v=vjU62r-M1CY

## Build and run
This project uses cmake as build system. Follow below instructions to run it.

Setup
```shell
cmake .
```
Build
```shell
cmake --build .
```
Run
```shell
./raylib-2d-rpg/raylib-2d-rpg
```
